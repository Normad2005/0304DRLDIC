# Jim Yang's Personal Professional Page

A minimalist, high-contrast, and dynamic personal professional page inspired by modern corporate design systems (such as the Hostinger Presson template).

## Features

- **Bold Typography:** Employs modern aesthetic with "Outfit" and "Inter" Google Fonts for high-impact headers and crisp, readable body text.
- **Dynamic Interactions:**
    - Custom cursor dot that smoothly follows the user's mouse and expands on interactive elements.
    - Subtle pulsing effects on structural elements like the time display.
- **Precise Live Time Display:** Continuously computes and updates the displayed time and date starting from a highly specific base time.
- **Smooth Animations:** Elements elegantly slide, fade, and scale upon loading or interaction to create a lively and polished user experience.
- **Asymmetrical Layouts:** Utilizes intentional whitespace, high-contrast monochrome banding, and abstract geometric shapes to form a sophisticated modern corporate look.

## Technologies Used

- **HTML5:** Semantic structure and elegant layout foundations.
- **Vanilla CSS3:** Custom styling, variables, keyframe animations, and custom cursor logic without dependencies on heavy frameworks.
- **Vanilla JavaScript:** Powers the live clock updates and smooth custom cursor positioning.

## Setup Instructions

1. Clone or download the repository to your local machine.
2. Open `index.html` directly in any modern web browser to view the site. No local server or build pipeline is required.
